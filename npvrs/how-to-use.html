Step-by-step guide with exact URLs and examples

1 Check System Status

First, verify your installation is working:

Open in browser: http://localhost/npvrs/status.php
‚úÖ All checks should be green. If not, follow the error messages to fix issues.

2 Import Database Schema

Method 1: Using phpMyAdmin (Easiest)

  1. Open: http://localhost/phpmyadmin
  2. Click "Import" tab
  3. Click "Choose File"
  4. Select: c:\xampp\htdocs\npvrs\database\schema.sql
  5. Click "Go"

Method 2: Using MySQL Command Line

cd c:\xampp\mysql\bin mysql -u root -p SOURCE c:/xampp/htdocs/npvrs/database/schema.sql;

3 Install Postman or Thunder Client

You need a REST API client to test the endpoints:

Download Postman OR Install "Thunder Client" extension in VS Code
üí° Tip: Import the included collection file: postman_collection.json

4 Test Your First API Call

Register an Administrator

POST Request:

http://localhost/npvrs/controllers/AdminController.php?action=register

Headers:

Content-Type: application/json

Body (raw JSON):

{ "full_name": "John Administrator", "email": "admin@hospital.com", "password": "Admin@123456" }

Expected Response (201 Created):

{ "success": true, "message": "Administrator registered successfully", "data": { "id": 1, "full_name": "John Administrator", "email": "admin@hospital.com", "created_at": "2025-10-29 12:00:00" } }

5 Complete Workflow Example

A. Register Hospital

POST http://localhost/npvrs/controllers/HospitalController.php?action=register

{ "hospital_name": "City General Hospital", "hospital_address": "123 Main Street, Springfield", "email": "contact@cityhospital.com", "phone_number": "1234567890", "password": "Hospital@123456" }
Status will be "pending" - Hospital cannot login yet!

B. View Pending Hospitals (as Admin)

GET http://localhost/npvrs/controllers/AdminController.php?action=pending-hospitals

C. Approve Hospital (as Admin)

POST http://localhost/npvrs/controllers/AdminController.php?action=approve-hospital

{ "hospital_id": 1, "admin_id": 1 }
‚úÖ Hospital status changes to "approved" and receives email notification!

D. Hospital Login (Now It Works!)

POST http://localhost/npvrs/controllers/HospitalController.php?action=login

{ "email": "contact@cityhospital.com", "password": "Hospital@123456" }

E. Register Patient (as Hospital)

POST http://localhost/npvrs/controllers/HospitalController.php?action=register-patient

{ "hospital_id": 1, "full_name": "Jane Doe", "date_of_birth": "1990-05-15", "gender": "female", "phone_number": "9876543210", "address": "456 Oak Avenue" }
‚úÖ Patient gets unique ID like: PID-20251029-00123

F. Add Medical Record (as Hospital)

POST http://localhost/npvrs/controllers/HospitalController.php?action=add-medical-record

{ "hospital_id": 1, "patient_id": 1, "diagnosis": "Common cold with mild fever", "prescription": "Rest and fluids for 3 days", "symptoms": "Cough, runny nose" }

6 All Available Routes

Admin Routes

Method Action Description
POST register Register new admin
POST login Admin login
GET pending-hospitals List pending hospitals
POST approve-hospital Approve hospital
POST reject-hospital Reject hospital

Hospital Routes

Method Action Description
POST register Register hospital (pending)
POST login Hospital login (if approved)
POST register-patient Register new patient
GET get-patients Get hospital's patients
POST add-medical-record Add diagnosis/prescription
GET get-medical-history Get patient medical history

7 Check Logs and Emails

All email notifications are logged to:

c:\xampp\htdocs\npvrs\logs\emails.log

Database errors are logged to:

c:\xampp\htdocs\npvrs\logs\database_errors.log
üí° Tip: Open these files in a text editor to see email content and debug issues.
